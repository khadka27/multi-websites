## 🏃‍♂️ Getting Started

1. **Install Dependencies**

```bash
npm install
```

2. **Run Development Server**

```bash
npm run dev
```

3. **Build for Production**

```bash
npm run build
```

4. **Start Production Server**

```bash
npm start
```

The application will be available at [http://localhost:3000](http://localhost:3000)
